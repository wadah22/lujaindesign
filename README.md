# Lujain Design | لجين ديزاين

A luxury abaya boutique website for **Lujain Design**, based in Aden, Yemen. The site showcases an exclusive collection of classic, embroidered, and evening abayas with direct ordering via WhatsApp and Instagram.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | TanStack Start (SSR) |
| Frontend | React 19, TanStack Router v1 |
| Build | Vite 7 |
| Styling | Tailwind CSS 4 + custom CSS variables |
| Language | TypeScript 5.7 (strict mode) |
| Deployment | Netlify |

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:3000` (or port 8888 via Netlify CLI).

```bash
# Using Netlify CLI (recommended for full platform feature emulation)
netlify dev
```

## Features

- RTL Arabic layout with luxury gold/black design aesthetic
- Product catalog with category filtering
- Product detail modal with direct WhatsApp ordering
- Contact form that opens WhatsApp with pre-filled message
- Responsive mobile layout
- Floating WhatsApp button
- Smooth scroll navigation
